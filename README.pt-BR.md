## Jogo da Velha

Um programa simples que permite jogar Jogo da Velha contra o computador.

## Como executar
```bash
python tic_tac_toe.py
```

## Funcionalidades
- Mudar os símbolos usados pelo jogador e computador através de variáveis globais no código
